このツールは？

ゲーム「Lobotomy Corporation」プレイ時のマネジメントを補助するツールです。
主に「職員を育成する際のアブノーマリティと職員の割当て」を考えるのを補助します。
個人で使っているオレオレツールを使う人いるかなーと公開してるレベルなので、 色々足りない部分があります。
また、プレイスタイルによっては役に立たないかもしれません。


そもそもこのツールを作ろうと思ったのは 職員の育成を考えながら配属することを考えた時、

  • 職員ごとの育成したい能力値に応じて、対応する作業の成功率が高いアブノーマリティに割り当てたい
  • 育成に有用なアブノーマリティを暇にしておきたくない
  • 1匹のアブノーマリティに1人の割当てにして事故を防ぎたい

等の条件を満たすように決めたいなーと思ったからです。
最初はメモ帳の上でアブノーマリティへの職員の割当作業メモを作っていましたが、 職員とアブノーマリティの数が増えるに連れ、ついうっかり割当を被らせてしまうなど、なかなか考えるのが難しくなってきました。

本ツールを使うことで

  • 収容しているアブノーマリティと作業適性を登録
  • 職員に割り当てる作業とアブノーマリティを登録
  • 登録結果を元にプレイ中に見やすい作業メモを生成

することでこの割当作業が簡単にできる…と思います。

使い方

1. 収容しているアブノーマリティを登録する

　

2. アブノーマリティの作業適性を登録する

　

3. 職員名を記入、作業とアブノーマリティを割り当てる

　

4. 作業メモを生成する
「エージェント」ウィンドウで「テキスト生成」ボタンを押すと作業メモ表示
※並び順はアブノーマリティの収容部門（施設上から）
※Starしたアブノーマリティへの作業未割り当て、職員への割当被りに応じて警告表示
